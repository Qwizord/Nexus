# Nexus Project Brain 🧠

Obsidian vault для контекста разработки приложения **Nexus** — AI-ассистент для роста в здоровье, финансах и обучении.

## 📊 Структура

- **[[Status]]** — текущий статус всех фич
- **[[Architecture]]** — техстек, MVVM, Firebase, n8n
- **[[Features]]** — описание каждого экрана и фичи
- **[[Issues]]** — активные баги и их статус
- **[[Design]]** — палитры, стили, Liquid Glass
- **[[Performance]]** — оптимизация, профилирование
- **[[Analytics]]** — метрики и события (планируется)
- **[[Roadmap]]** — что дальше после MVP

## 🎯 Быстрый старт

Перед каждой сессией с Claude:
1. Обнови [[Status]]
2. Добавь баги в [[Issues/Active]]
3. Запиши решения в [[Solutions]]
